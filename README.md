# backup-pi

## Installation

Download script:
```
    wget https://github.com/Takabuu/backup-pi/blob/main/backup-pi.sh
```

Make executable:
```
    chmod +x backup-pi.sh
```

You are ready to backup your Pi! :bowtie:
